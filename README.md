# 📊 End-to-End Sales Analysis

## 📖 Overview
This project performs **end-to-end sales data analysis** using Python and Pandas.  
It covers the full workflow from data loading to generating insights and reports.

---

## 🎯 Objective
- Analyze sales data  
- Calculate total revenue and trends  
- Identify best-selling products  
- Generate a clean report with insights  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- VS Code  

---

## 📅 Project Workflow

### ✅ Day 1: Setup & Load Data
- Installed pandas library  
- Created Python script  
- Loaded CSV dataset  

### ✅ Day 2: Explore Data
- Viewed dataset (`head()`)  
- Checked shape (rows, columns)  
- Viewed column names  
- Checked data types (`info()`)  
- Generated statistics (`describe()`)  
- Identified missing values  

### ✅ Day 3: Clean Data
- Handled missing values  
  - Numeric → filled with 0  
  - Categorical → filled with "Unknown"  
- Removed duplicate rows  
- Fixed data types  
- Saved cleaned dataset  

### ✅ Day 4: Analyze Sales
- Calculated total revenue  
- Found average, highest, lowest sales  
- Identified best-selling product  
- Generated product-wise summary  
- Extracted top 5 products  

### ✅ Day 5: Create Report
- Generated formatted report  
- Included key metrics and insights  
- Saved report as `.txt` file  

---

## 📊 Key Features
- Data cleaning and preprocessing  
- Sales performance analysis  
- Automated report generation
